# ai_interview_gen

## Clone Repository 
To clone repository: `git clone https://github.com/nhombs035/ai_interview_gen.git

AcCess repository in your directory: `cd ai_interview_gen`

## Create a Virtual Environment 
To create a virtual environment: `python -m venv venv`
To activate virtuual environtment in Windows: `venv\Scripts\activate`

## Install Dependencies 
Install Dependencies: `pip install -r requirements.txt`


## Add API Key
PLEASE NOTE This requires an API key, to run withought purchasing your own API key, please see the deployed website:
[AI Interview Simulator Website](https://aiinterview-simulator.streamlit.app/) 

If adding your own API key create an .env and add:
`OPEN_API_KEY = your_api_key`

## Run Application
To run application: `streamlit run main.py`
